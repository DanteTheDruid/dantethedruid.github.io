Spawn System

Interactive mob spawning system with progression-based unlocks and balanced token costs. Viewers can spawn creatures that match the current game progression level.

Mob Tiers

Harmless: Peaceful critters and non-hostile mobs
Examples: Bunny, goldfish, bird, squirrel, penguin, frog, duck, firefly, butterfly, snail, corrupt goldfish

Common: Basic slimes and weak enemies
Examples: Slimes (blue, green, purple, yellow, red, jungle, ice), zombie, demon eye, cave bat, eater of souls, crab, piranha, corrupt bunny

Rare: Stronger pre-hardmode enemies, uncommon hardmode enemies, and more
Examples: Skeletons, hornets, jungle/dungeon/goblin enemies, unicorn, pixie, hardmode enemies, etc.

Deadly: The most dangerous non-boss enemies, mini-bosses, and rare hardmode threats
Examples: Wraith, gastropod, possessed armor, corruptor, clinger, werewolf, mimic, spiked ice slime, deadly jungle/ocean/cavern enemies, etc.

Bosses are not included in these tiers and are spawned via a separate command.

Important Notes

  • Tiered Spawning: Use !spawn <tier> to spawn a random mob from a tier filtered by your progression. Tiers: harmless, common, rare, deadly.
  • Progression-Based: Mobs unlock as you defeat bosses and progress through the game. Progression stages: Early Game, Post-Eye, Post-Brain/Eater, Post-Skeletron, Hardmode, Post-Mechanical, Post-Plantera.
  • Smart Positioning: The system tries up to 40 times to find a valid spawn location near a player, using smart logic to avoid blocks and ensure mobs appear in accessible areas. If no perfect spot is found, it will use the best fallback, and as a last resort, spawn above the player with a warning.
  • Progression Gating: Only mobs appropriate for your world state can be spawned (unless you are a Game Master).
  • Token Costs & Mob Lists: Both are configurable in the mod settings. If no mobs are available for a tier, the bot will notify you.
  • Game Master Privileges: Streamers and moderators can spawn any mob for free.
  • Auto-Despawn: Spike Balls automatically despawn after 60 seconds (unkillable).
  • Complete List: Over 100+ different spawnable creatures available.

Setup Guide

Get TwitchIntegrationPlus running in just 3 simple steps

1

Install & Configure

Enable the mod in tModLoader and set your channel name in Mod Configuration

  • Download and install the mod from the mod browser (Steam Workshop coming soon)
  • Launch Terraria with tModLoader
  • Go to Mods → Mod Configuration → TwitchIntegrationPlus
  • Set your Twitch channel name
2

Authorize

Use the /twitch auth command in-game to connect your Twitch account

  • Open the in-game chat
  • Type /twitch auth
  • Follow the Twitch authorization flow
3

Connect

Use /twitch connect to join your Twitch chat and start streaming!

  • Type /twitch connect in-game
  • The mod will connect to your Twitch chat
  • Viewers can now use commands in your chat
  • Check status with /twitch status
