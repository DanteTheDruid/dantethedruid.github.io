Setup Guide

Get TwitchIntegrationPlus running in just 3 simple steps

1

Install & Configure

Enable the mod in tModLoader and set your channel name in Mod Configuration

  • Download and install the mod from the mod browser (Steam Workshop coming soon)
  • Launch Terraria with tModLoader
  • Go to Mods → Mod Configuration → TwitchIntegrationPlus
  • Set your Twitch channel name
2

Authorize

Use the /twitch auth command in-game to connect your Twitch account

  • Open the in-game chat
  • Type /twitch auth
  • Follow the OAuth authorization flow
  • Your tokens will be securely encrypted and stored
3

Connect

Use /twitch connect to join your Twitch chat and start streaming!

  • Type /twitch connect in-game
  • The mod will connect to your Twitch chat
  • Viewers can now use commands in your chat
  • Check status with /twitch status

Required Permissions

The mod requests the following OAuth scopes for full functionality:

chat:read Read chat messages
chat:edit Send chat messages
channel:read:subscriptions Monitor subscriptions
bits:read Monitor bit donations
channel:read:redemptions Monitor channel point redemptions
moderator:read:followers Monitor new followers

Security Features

Built with security and streaming safety in mind

Encrypted Token Storage

OAuth tokens are encrypted and stored separately from configuration files, ensuring maximum security.

UI Hidden

OAuth tokens never appear in the configuration interface, preventing accidental exposure.

Streaming Safe

No risk of accidentally showing tokens during streams - everything is hidden by design.

Secure by Default

Built-in Client ID is safe for public distribution - no need to create your own Twitch app.

Token Validation

Automatic validation and refresh handling ensures uninterrupted streaming experience.
